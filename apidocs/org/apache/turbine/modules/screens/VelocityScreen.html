    public class VelocityScreen
extends TemplateScreen
    Base Velocity Screen. The buildTemplate() assumes the template parameter has been set in the PipelineData object. This provides the ability to execute several templates from one Screen.

    If you need more specific behavior in your application, extend this class and override the doBuildTemplate() method.
